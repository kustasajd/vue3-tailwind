# Akara Portal

It was built with Vite, Vue 3, Tailwind CSS and TypeScript.

## Project setup
```
1. npm i
2. set apiUrl in config.js file
```

## Compiles and hot-reloads for development
```
npm run dev
```

## Compiles and minifies for production
```
npm run build
```
